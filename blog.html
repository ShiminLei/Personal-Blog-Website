Attribute-Controlled Dialog Generation System based on Hybrid Learning Object

November 17, 2019 by Shimin Lei

Learnt Dialog Act (DA) Recognition Network (DARN) using Discrete Variational Auto-encoder with Gumbel Sampling. Built Dialog Generation System composed of Policy Network, Hierarchical Recurrent Encoder and GRU-RNNs Decoder. Trained the system using supervised learning and reinforcement learning (self-critical sequence training) with DARN. Solved the posterior collapse more efficiently with 52.53 Perplexity, 0.13 KL divergence, and 1.18 mutual information in DARN. Achieved 0.72 higher readability and 0.51 higher BLEU score for Dialog Generation System.

Operating Room Time Optimization of UVA Main Facility

October 23, 2019 by Shimin Lei

Evaluated and optimized the Operating Room time utilization of UVA main facility with statistical modeling. Used ggplot2 to visualize the data, and did time series analysis to explore the overutilization problem. Used Structural Equation Model to select the best model for fitting multicollinearity of time data. Predicted the best management scheme with Monte Carlo Simulation to decrease the variability of overutilized time. Built 3D system simulation model with Simio to verify the scheme, and achieved 10% decrease in overutiliztion.

Neural Text Generation Model with Consistent Personality

September 5, 2019 by Shimin Lei

Built chatbots which can keep the consistency of different character speaking styles in the comedy South Park. Crawled the dialogues on the Internet with Scrapy. Used Glove for word embedding, and designed the persona vector. Pre-trained the persona-based text generation model with the global attention mechanism, beam search, scheduled sampling. Fine-tuned the base model by different characters, got different persona vectors by unfreezing them in self-designed LSTM. Compared different character models with BLEU score, and the best achieved score of 0.17 on average and 0.75 the best.
